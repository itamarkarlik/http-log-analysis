# Findings

## Initial HTTP Traffic Baseline

Initial HTTP log analysis was performed to understand the size, structure, and general behavior of the dataset.

The dataset contains 2,048,440 HTTP events.

Baseline analysis included HTTP event volume over time, top source IPs, HTTP status code distribution, HTTP method distribution, and top requested URIs.

At this stage, the queries provided general visibility into the dataset but did not independently confirm malicious activity.
